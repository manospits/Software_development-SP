Γενικές σημειώσεις εκτέλεσης:
-Μεταγλώττιση: make
(Σε περίπτωση που εμφανιστεί error κατά τη μεταγλώττιση, δείτε παρακάτω περί unit testing. Το εκτελέσιμο της εργασίας έχει δημιουργηθεί και τρέχει δίχως πρόβλημα.)
-Το πρόγραμμα εκτελείται με δύο ορίσματα, ως εξής:
./spath <αρχείο αρχικού γράφου> <αρχείο workload>
-Τα αποτελέσματα του προγράμματος βγαίνουν στο αρχείο results.txt που δημιουργείται
στον ίδιο φάκελο με το εκτελέσιμο.

Σημειώσεις περί Unit testing:
-Στην άσκηση έχουν συμπεριληφθεί αρχεία για unit testing.
Το unit testing που επιλέχθηκε χρησιμοποιεί το πρόγραμμα check.
Για να μεταγλωττιστούν τα αρχεία απαιτείται να είναι εγκατεστημένο το πρόγραμμα,
αλλιώς ΒΓΑΙΝΕΙ ΛΑΘΟΣ ΚΑΤΑ ΤΗΝ ΜΕΤΑΓΛΩΤΤΙΣΗ (καθώς λείπει το αντίστοιχο header file).
-Για να εκτελέσετε το unit testing (δεδομένου ότι υπάρχει το απαιτούμενο πρόγραμμα
που αναφέρεται παραπάνω, και έχει γίνει η μεταγλώττιση), τρέξτε:
./unittesting
